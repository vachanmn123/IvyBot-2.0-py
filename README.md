## Running the Bot

To run the bot, follow the steps below

1. install the requirements:
```
$ pip install -r requirements.txt
```

2. Edit the config files:
```
$ cp config.json.example config.json
$ nano config.json
```

3. Run the bot:
```
$ python3 main.py
```
